# Estimate

## Getting Started

Welcome to Estimate! This guide will help you set up and start using the project quickly.

---

## Project Information

### Name

Estimate

### Description

Estimate is a project aimed at providing accurate and efficient time estimations while analyzing various factors that impact productivity. It is designed to streamline project planning and improve time management.

---

## Installation

To install and run Estimate, follow these steps:

1. **Clone the Repository**  
   Open your terminal and run:

   ```bash
   git clone https://ice.cs.ndsu.nodak.edu/winners/estimate.git
   ```

2. **Import into Eclipse IDE**

   - Open Eclipse.
   - Import the cloned repository as a Maven project.

3. **Run the Project**
   - Use `jetty:run` to launch the project:
     1. Open the Maven view in Eclipse.
     2. Navigate to the `jetty` plugin.
     3. Execute the `jetty:run` goal.
